# Venus Gecko 참조 사이트 분석

## 시각적 특징
- **배경**: 크레스티드 게코와 자연 환경(나뭇가지, 이끼)의 고품질 사진
- **색상 팔레트**: 
  - 주요: 흰색(텍스트), 황금색/베이지(강조)
  - 배경: 자연 톤(갈색, 녹색)
- **헤더**: 검정색 배경, 흰색 텍스트, 황금색 Contact 버튼
- **타이포그래피**: 세리프 폰트(제목), 산세리프 폰트(본문)

## 페이지 구조
1. **네비게이션**: HOME, ABOUT, SIGNATURE, VISIT + 언어선택(KR) + Contact 버튼
2. **히어로 섹션**: 게코 이미지 배경, 큰 제목("Venus Gecko"), 부제("VENUS GECKO")
3. **About 섹션**: "Eye-contact addicts" 메시지, 설명 텍스트
4. **Core Essence**: 3가지 핵심 가치 (Pure Quality, Artistic Care, Premium Trust)
5. **Signature Line**: 게코 카드 슬라이더 (MALE/FEMALE 필터, 좌우 네비게이션)
6. **Visit 섹션**: 매장 위치 정보, 예약 링크, SNS 링크
7. **푸터**: 저작권, 링크들

## 주요 기능
- 언어 선택 (KR)
- 게코 필터 (MALE/FEMALE)
- 게코 카드 슬라이더 (Embla Carousel)
- 반응형 디자인

## 제외할 기능
- 매장 위치 지도 (VISIT 섹션의 지도 기능)
- 예약 기능 (BOOK A VISIT)

- 고유한 색상 팔레트

- 추가적인 인터랙션 요소
